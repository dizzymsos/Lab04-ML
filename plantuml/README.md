# Modelo de clases PlantUML

Archivo principal:

```text
plantuml/modelo_clases_lab04.puml
```

Para renderizarlo con PlantUML instalado localmente:

```bash
plantuml plantuml/modelo_clases_lab04.puml
```

También se puede abrir el archivo `.puml`, copiar su contenido y usarlo en el sitio oficial de PlantUML:

```text
https://plantuml.com/es/
```
